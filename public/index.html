Welcome on OTO's backend

Here are all the routes

/connect GET -- Sends user to the Spotify authorization website

/releases GET -- When user is redirected his acces token en refresh token is set, when done user is redirected to the website

/getuser GET -- Gets all user profile information

/currentsong GET -- Gets users currentsong

/newreleases GET -- Gets new releases for Belgian market

/allgenerated GET -- Gets all past generated playlist id's and transforms it into playlists

/featured POST -- Gets all user choices and makes a recommended playlist

/create POST -- Creates an empty playlist on Spotify with users chosen name

/addtoplaylist POST -- Gets generated songs from database and adds them to the empty playlist

/lastplaylist GET -- Gets the created playlist of the user and sends the result
